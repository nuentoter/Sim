Project: Gull Island

Genre:
Text-based investigative simulation.

Core Idea:
The player is a private investigator on an island community.

Architecture:
- Flask API
- NPC registry
- Truth layer
- Rumor layer
- Belief layer
- Investigation board
- Salience system
- Scenario system

Design Goals:
- Noir sandbox
- Economy matters
- Social status matters
- NPCs have lives independent of player
- Cases emerge from daily activity

Current Status:
Generation 1 complete.
